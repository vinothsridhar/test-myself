## The deepest insight

You can think of a computer booting like this:

```
Physics layer
Electricity flows
```

↓
```
Oscillation layer
Quartz vibrates
```
↓
```
Clock layer
Tick Tick Tick
```
↓
```
Hardware state machines
CPU wakes up
```
↓
```
Firmware
BIOS/UEFI runs
```
↓
```
Operating System
Linux / Windows
```
↓
```
Applications
Chrome
Java
Postgres
```

Everything above ultimately depends on that tiny crystal starting to oscillate when power is applied. That's the heartbeat that brings the entire computer to life.
